# COVID-19 Data Analysis 📊

This project analyzes global COVID-19 data to understand trends in cases, deaths, and recoveries across different continents.

## Tools Used
Python, Pandas, Matplotlib, Seaborn

## Key Analysis
- Continent-wise cases and deaths
- Correlation between cases, deaths, and recoveries
- Data visualization using charts and heatmaps

## Insights
- Strong correlation between cases and deaths
- Europe, Asia, and North America are most affected
- Recovery trends improved over time
